MW Theme URI Shortcode to make a shortcord outputting theme directory uri.

== Description ==

MW Theme URI Shortcode to make a shortcord outputting theme directory uri.
shortcode : [template_directory_uri], [stylesheet_directory_uri]

When you switches Visual to Text,  replaced into shortcode.
When you switches Text to Visual, shortcode replaced into theme uri.
IF you don't use child theme, there are cases where [stylesheet_directory_uri] replaces [template_directory_uri]

== Installation ==

1. Upload `MW Theme URI Shortcode` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
